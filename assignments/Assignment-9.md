### Part A
1. **Perceptron:** A simple, single-layer artificial neuron that takes numerical inputs, applies weights, and uses a step function for binary classification.
2. **Key differences (Perceptron vs DNN):** 
   - 1 layer vs multiple hidden layers.
   - Linear problems vs non-linear problems.
   - Step function vs advanced activation functions (ReLU, Sigmoid).
3. **Gradient Descent:** Optimization algorithm that adjusts network weights to minimize prediction error.
4. **Backpropagation:** Calculates how much each weight contributed to the error, allowing Gradient Descent to adjust them correctly.
